# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do **not** open a public issue. Instead, please report it via one of the following methods:

1. **GitHub Security Advisory**: Use the [GitHub interface](https://github.com/dakotaveaux/mRNA/security/advisories/new) to report a private vulnerability.
2. **Email**: Send a detailed report to dakotaveaux@gmail.com.

We aim to acknowledge all reports within 48 hours and provide a resolution or mitigation plan within 1 week.
